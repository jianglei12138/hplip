    • Upgrading HPLIP


    Upgrading HPLIP is done by first removing the current installed version of HPLIP installed and then installing the newer version of HPLIP.

    Remove HPLIP by either running "make uninstall" in the HPLIP source directory or by running "rm -rf /usr/share/hplip".

    If you have a preinstalled version of HPLIP you can uninstall it by using the package manager that your system uses.

    Install the newer version of HPLIP by downloading and then running the automatic installer or by using the manual steps. More information on downloading and installing HPLIP is available on our website.
