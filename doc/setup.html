    • Setting up a new printer with HP Device Manager Setup Wizard (hp-setup)

    Configuring your printer can be done several ways.

    Configure using hp-toolbox

    1. Open a terminal (command line) window.

    2. Run:

                hp-toolbox

    3.  If no printer has been configured, the "No Installed HP Devices Found" dialog will appear.  Click "Setup Device..."

    4.  "Choose connection type" will be displayed.  Here select the way you want to connect to the printer:  USB, Network, or Parallel

    5.  Next, the dialog "Select from Discovered Devices" will display the detected printers.  If you are configuring a network printer you may need to select "Find Manually..".  

    Note:  If your printer is not displayed, verify that the printer is connected, powered on and not in an error state (i.e. low ink, out of paper, etc)

    6.  The "Select/Confirm PPD file" dialog is now presented.  The file needs to match the model/class of your printer.

    7.  "Enter Printer Information' -- Here you can enter the Printer name, location and other information.

    8.  Finally, the "Finished Adding Printer" dialog is displayed.  Verify the printer information and click "Finish" to complete the configuration of your printer.

    2. Run:

    Note:  If your printer is not displayed verify that the printer is connected, powered on and not in an error state (i.e. low ink, out of paper, etc)

    5.  The "Select/Confirm PPD file" dialog is now presented.  The file needs to match the model/class of your printer.
