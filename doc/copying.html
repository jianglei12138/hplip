HP Linux Imaging and Printing (HPLIP) Help

  • Main Menu

    • Copying (hp-makecopies)


    If your device supports making copies you can use the hp-makecopies application to make copies.  This feature is accessed either by running hp-toolbox, click the Actions tab and then click make copies, or by running hp-makecopies from the command line.  Not all printing devices support this feature.
