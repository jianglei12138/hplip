    • PC Send Fax (hp-sendfax)


    hp-sendfax is used to access the fax features of your supported device.  To access hp-sendfax you can either run hp-toolbox, click the fax queue and then click send fax or run hp-sendfax from the command line.

    Using hp-sendfax is rather straight forward.  Add the file, add the phone number to "fax to", and then click send fax now.
